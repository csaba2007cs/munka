# Atlas referencia — helyőrző

A megrendelői **Atlas** vizuális referenciakép **még nem** került a repóba.

Amikor megérkezik:

1. Mentsd ide: `shared/assets/images/atlas-reference.png` (vagy `.webp`).
2. Frissítsd a token értékeket [shared/css/theme.css](../shared/css/theme.css) fájlban a [design-tokens.md](design-tokens.md) útmutatása szerint.
3. Opcionális: rövid megjegyzés ebben a fájlban a dátumról és a változtatott színekről.

Addig a UI a meglévő Nanoportal sci-fi témát használja (Orbitron / Rajdhani / Roboto Mono).
